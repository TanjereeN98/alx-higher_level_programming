Usage
----------
MyList Class has a public instace method: def print_sorted(self): that prints the list, but sorted (ascending sort)

::

>>> MyList = __import__('1-my_list').MyList
>>> lst = MyList()

::

>>> lst.print_sorted()
[]

::

>>> lst = MyList([13, 1, 0, 3, -2, 5])
>>> lst.print_sorted()
[-2, 0, 1, 3, 5, 13]

::

>>> lst = MyList(['g', 'a', 'z', 'b'])
